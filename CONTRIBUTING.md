# Contributing Guide

Thanks for contributing to Runtime Visualizer Lab.

Repository: https://github.com/mhmdrameez/runtimevisulizerlab

## Development Setup

```bash
pnpm install
pnpm dev
```

## Branch Naming

- `feat/<short-description>`
- `fix/<short-description>`
- `docs/<short-description>`
- `refactor/<short-description>`

## Commit Style

Use clear commit messages:

- `feat: add runtime output verification marker`
- `fix: prevent mobile editor collapse`
- `docs: update contribution guide`

## Pull Request Checklist

- Code is focused and minimal
- `pnpm lint` passes
- UI changes are responsive on mobile and desktop
- README/docs updated when behavior changes
- PR includes short summary and screenshots for UI changes

## Reporting Issues

When opening an issue, include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Browser/OS details
- Console errors (if any)
