# Runtime Visualizer Lab (INSIDE JS)

Interactive JavaScript runtime visualizer to understand:
- Call Stack
- Event Loop
- Microtasks / Macrotasks
- Memory Heap
- Runtime-verified `console.log` output

Repository: https://github.com/mhmdrameez/runtimevisulizerlab

## Features

- Step-by-step execution timeline
- Monaco editor with live error markers
- Background runtime verification (real output vs simulated output)
- Array/object memory breakdown in simulation
- Mobile-responsive editor and panels
- SEO setup (`sitemap.xml`, `robots.txt`, Open Graph, JSON-LD)

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Monaco Editor

## Run Locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
pnpm lint
```
